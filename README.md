## Compatibility

Minesweeper is compatible with Windows, Mac OS, and Linux.

## Setup

``` bash
# clone the repository
git clone https://github.com/Tellardis/electron-minesweeper.git

# install dependencies
npm install

# start))
npm run start

# build electron application
npm run dist
```
